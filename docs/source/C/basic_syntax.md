## 有符号和无符号数

**`~x = -(x + 1)`**  
* -8的机器表示：-8 -> -(7+1) -> ~7 -> ~(0111) -> 1000  
* -5的机器表示：-5 -> -(4+1) -> ~4 -> ~(0100) -> 1011

* 永远不要为了多一个二进制的精度而用无符号表示数值
* 循环溢出问题：由于`++`运算会进行模运算， 因此：
    + 无符号：溢出后会变为0，从而循环终止；
    + 有符号：溢出后变为最小负数，然后逐渐增大到0，循环终止
* 整数溢出时，一般策略为丢弃最高有效位(即最高位)
